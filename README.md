# Weather-CLI

Aplicativo de linha de comando em Python para consultar previsões do tempo usando a [BrasilAPI](https://brasilapi.com.br/).  
Ele exibe as condições do tempo com emojis, salva um histórico local em arquivo JSON e oferece uma interface simples e interativa.

---

## Funcionalidades

- Consulta a previsão do tempo por cidade no Brasil.  
- Exibe dados de mínima, máxima, condição e data.  
- Histórico local de consultas salvas em arquivo JSON.  
- Ícones visuais para clima (chuva, ensolarado, nublado).  
- Interface de menu no terminal fácil de usar.
